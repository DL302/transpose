# transpose
Python script to automate transposing a column of data in "blocks"

## Example

|             | A           | B           | C           | D           | E           | F           |
| ----------- | ----------- | ----------- | ----------- | ----------- | ----------- | ----------- |
| **1**           | Z           | 20          |             |             |             |             |
| **2**           | Z           | 21          |             | Z           | 20          | 21          |
| **3**           | Y           | 22          |             | Y           | 22          | 23          |
| **4**           | Y           | 23          |             | X           | 24          | 25          |
| **5**           | X           | 24          |             |             |             |             |
| **6**           | X           | 25          |             |             |             |             |

To the left is the original data and to the right is the formatted data
